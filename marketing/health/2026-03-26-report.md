## Top 3 Marketing Highlights This Week

### 1. Cognitive Load Detection — A Research-Backed Differentiator
OpenTutor ships 12-signal behavioral cognitive load detection backed by a 2025 Nature paper. No other open-source learning tool does this. The system monitors fatigue, error patterns, and message brevity to adapt tutor depth in real time. This is a headline-worthy differentiator that is currently buried in the roadmap checklist — it deserves its own README section and a standalone blog post.

### 2. LOOM + LECTOR Knowledge Graph System (Experimental but Live)
Two 2025 arxiv papers are implemented and running: LOOM (dynamic learner memory graph) and LECTOR (semantic spaced repetition via prerequisite ordering). These are not vaporware — they're in the codebase with dedicated test files. The "experimental" label undersells what's there. Highlighting the academic grounding will resonate with researchers, educators, and HackerNews audiences.

### 3. Zero-API-Key Local-First Setup in 3 Commands
The 3-command Docker quick-start (`git clone` → `cp .env.example .env` → `docker compose up`) with Ollama as the default LLM provider is a strong privacy/sovereignty hook. No credit card, no cloud account, no data leaves the machine. This is the single most shareable feature for privacy-conscious communities (r/selfhosted, Lobsters, HN).
