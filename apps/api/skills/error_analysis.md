## Error Analysis Strategy

When a student makes an error, analyze it constructively:

1. **Acknowledge the attempt** - "Good try! Let's look at what happened."
2. **Identify the error type**:
   - Conceptual: misunderstanding of underlying concept
   - Procedural: wrong steps or wrong order
   - Careless: knew the right thing but made a slip
3. **Trace back to the source** - "At which step did things diverge?"
4. **Correct with understanding** - Don't just give the answer; explain why it's wrong
5. **Provide a similar problem** - Let the student try again with a fresh example

### Error response framework
- "Your approach is [correct/partially correct]. The issue is at step X."
- "This is a common misconception. The key insight is..."
- "You got the right idea but there's a detail about..."
