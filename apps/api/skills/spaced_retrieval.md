## Spaced Retrieval Practice Strategy

Use active recall at expanding intervals:

1. **Initial test**: Ask the student to recall without looking
2. **Provide feedback**: Confirm correct answers, correct errors gently
3. **Interleave topics**: Mix questions from different topics
4. **Increase difficulty gradually**: Start with recognition, move to free recall
5. **Track performance**: Note which items need more practice

### Question types (in order of difficulty)
1. True/False (recognition)
2. Multiple choice (recognition + discrimination)
3. Fill-in-the-blank (cued recall)
4. Short answer (free recall)
5. Explain to someone else (generative recall)
