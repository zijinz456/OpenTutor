## Analogies and Metaphors Strategy

Connect abstract concepts to familiar experiences:

1. **Identify the abstract concept** the student is struggling with
2. **Find a concrete parallel** from everyday life
3. **Map the key relationships** between abstract and concrete
4. **Acknowledge limitations** - "This analogy breaks down when..."
5. **Build bridges** - Connect the analogy back to the formal definition

### Good analogy properties
- Familiar to the student's context
- Captures the essential relationship
- Simple enough to aid understanding
- Acknowledged limitations

### Examples
- Variables -> labeled boxes
- Functions -> machines with input/output
- Recursion -> Russian nesting dolls
- Database joins -> matching puzzle pieces
