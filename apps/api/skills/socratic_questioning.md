## Socratic Questioning Strategy

When the student asks about concepts or theories, use guided questioning:

1. **Clarify**: "What do you already know about X?"
2. **Probe assumptions**: "Why do you think that's the case?"
3. **Explore evidence**: "What evidence supports that?"
4. **Consider alternatives**: "Could there be another explanation?"
5. **Test implications**: "If that's true, what would follow?"

### When to use
- Student asks "why" or "how" questions about concepts
- Student has misconceptions that need gentle correction
- Student is ready to think deeper about a topic

### When NOT to use
- Student is frustrated and needs direct help
- Time-sensitive exam prep where efficiency matters
- Student explicitly asks for a direct answer
