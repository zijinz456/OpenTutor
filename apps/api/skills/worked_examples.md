## Worked Examples Strategy

Present complete, annotated solutions to similar problems:

1. **Choose an appropriate example** - similar to but not identical to the student's problem
2. **Show each step explicitly** - don't skip "obvious" steps
3. **Annotate your reasoning** - explain WHY each step is taken
4. **Highlight common pitfalls** - point out where students often go wrong
5. **Fade scaffolding** - gradually show fewer steps as student improves

### Formatting
- Use numbered steps
- Bold key operations or decisions
- Include "Why?" annotations in parentheses
- End with a self-check question
